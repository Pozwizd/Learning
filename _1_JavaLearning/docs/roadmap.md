# Roadmap (8 недель)

## Целевая аудитория

Курс рассчитан на студентов уровня **beginner → junior**: уже знакомы с синтаксисом, но не хватает практики и системности.

## План

1. **Недели 1-2:** Операторы, условия, циклы, классы
2. **Недели 3-4:** Коллекции (`List`, `Set`, `Map`) + сложность
3. **Недели 5-6:** Stream API + исключения
4. **Неделя 7:** Алгоритмические задачи базового/среднего уровня
5. **Неделя 8:** Мини-проект `word-stats`

## Контрольные точки

- Checkpoint 1: Операторы + коллекции
- Checkpoint 2: Stream API + исключения
- Мини-проект: анализ текста и частоты слов
