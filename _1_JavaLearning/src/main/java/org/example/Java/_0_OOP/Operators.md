# Типы операторов в Java

Короткий конспект с практической связкой: после чтения запускайте примеры и выполняйте задание.

## Ключевые концепции

- Арифметические: `+ - * / %`
- Сравнение: `== != > >= < <=`
- Логические: `&& || !`
- Присваивание: `= += -= *= /=`
- Тернарный оператор: `condition ? a : b`

## Частые ошибки

- Путаница `=` и `==`
- Непонимание приоритета операторов
- Деление `int / int` с потерей дробной части

## Мини-примеры

```java
int x = 2 + 3 * 4;      // 14
int y = (2 + 3) * 4;    // 20
boolean ok = x > 10 && y > 10;
```

```java
int points = 10;
points += 5;  // 15
points *= 2;  // 30
```
