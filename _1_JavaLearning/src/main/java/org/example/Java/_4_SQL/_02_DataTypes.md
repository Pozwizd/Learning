# 2. Типы данных (Data Types)

Правильный выбор типа данных критичен для производительности и целостности данных.

## Числовые типы (Numeric)
Используются для хранения чисел.

| Тип | Описание | Пример (PostgreSQL/MySQL) |
| :--- | :--- | :--- |
| **INT / INTEGER** | Целое число (обычно 4 байта). | `123`, `-45` |
| **BIGINT** | Большое целое число (8 байт). | `9223372036854775807` |
| **SMALLINT** | Малое целое число (2 байта). | `32000` |
| **DECIMAL(p, s)** | Точное дробное число. `p` - общее кол-во цифр, `s` - после запятой. Идеально для денег. | `DECIMAL(10, 2)` -> `12345.67` |
| **FLOAT / REAL** | Число с плавающей точкой (приближенное). | `3.14159` |

## Строковые типы (String)
| Тип | Описание |
| :--- | :--- |
| **CHAR(n)** | Строка фиксированной длины. Если строка короче `n`, она дополняется пробелами. |
| **VARCHAR(n)** | Строка переменной длины (до `n` символов). Экономит место. |
| **TEXT** | Строка неограниченной длины (в PostgreSQL). В MySQL есть `TINYTEXT`, `TEXT`, `LONGTEXT`. |

## Дата и Время (Date/Time)
| Тип | Описание | Формат (ISO 8601) |
| :--- | :--- | :--- |
| **DATE** | Только дата. | `'2023-10-25'` |
| **TIME** | Только время. | `'14:30:00'` |
| **DATETIME / TIMESTAMP** | Дата и время. `TIMESTAMP` часто хранит время в UTC. | `'2023-10-25 14:30:00'` |
| **INTERVAL** | Промежуток времени. | `'1 day'`, `'2 hours'` |

## Логический тип (Boolean)
| Тип | Описание |
| :--- | :--- |
| **BOOLEAN** | Принимает значения `TRUE`, `FALSE` или `NULL`. |

## Двоичные данные (Binary)
| Тип | Описание |
| :--- | :--- |
| **BLOB** | Binary Large Object. Для хранения изображений, файлов. |
| **VARBINARY** | Аналог VARCHAR для байтов. |

## Специальные типы
*   **JSON / JSONB**: Для хранения JSON-документов (PostgreSQL имеет мощную поддержку `JSONB`).
*   **UUID**: Уникальный идентификатор (GUID).
*   **ARRAY**: Массивы значений (PostgreSQL).
*   **GEOMETRY**: Геометрические данные (PostGIS).

## Пример создания таблицы с разными типами
```sql
CREATE TABLE Employees (
    ID INT PRIMARY KEY,
    FullName VARCHAR(100),
    BirthDate DATE,
    Salary DECIMAL(15, 2),
    IsActive BOOLEAN,
    Metadata JSON
);
```
