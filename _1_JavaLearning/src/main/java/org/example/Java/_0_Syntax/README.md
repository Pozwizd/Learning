# _0_Syntax

Этот раздел предназначен для человека, который только начинает знакомство с программированием и Java.
Здесь собраны темы, которые стоит пройти до ООП: как запускается программа, как объявлять переменные, как работают условия, циклы, методы, массивы и строки.

Рекомендуемый порядок изучения:

1. `01_First_Program.md` - как выглядит программа на Java и что делает `main`
2. `02_Variables_And_Types.md` - переменные, примитивы, `String`, приведение типов
3. `03_Conditions.md` - `if`, `else`, `switch`
4. `04_Loops.md` - `while`, `do-while`, `for`, `for-each`
5. `05_Methods.md` - параметры, возвращаемые значения, `void`
6. `06_Arrays_And_Strings.md` - массивы, строки и базовые операции с ними
7. `Operators.md` - операторы Java и приоритет операций

Что важно понять в этом разделе:

- программа выполняется сверху вниз
- значения хранятся в переменных
- условия выбирают ветку выполнения
- циклы повторяют действия
- методы позволяют не дублировать код
- массивы и строки нужны для работы с наборами данных и текстом
