# 002. Машинная математика, переменные и типы данных

## Зачем это знать

Компьютер хранит данные в битах, а Java дает удобные типы, чтобы работать с этими данными безопасно и предсказуемо.

## Системы счисления

- двоичная система использует только `0` и `1`
- шестнадцатеричная компактно записывает двоичные данные
- в Java шестнадцатеричные литералы пишутся так: `0xFF`

## Базовые единицы

- `1 бит` хранит одно из двух состояний
- `1 байт = 8 бит`
- `int` занимает 4 байта
- `long` занимает 8 байт
- `double` хранит число с плавающей точкой двойной точности

## Основные типы Java

- целые: `byte`, `short`, `int`, `long`
- вещественные: `float`, `double`
- логический: `boolean`
- символьный: `char`

## Что важно на практике

- тип определяет диапазон значений и объем памяти
- переполнение целых типов не вызывает исключение, значение "перекручивается"
- `float` и `double` не подходят для точных денежных расчетов
- `char` хранит символ в Unicode

## Техническая часть

```java
int age = 25;
long population = 8_000_000_000L;
double price = 19.99;
char letter = 'J';
boolean ready = true;
```

## Типичные ошибки

- забыть суффикс `L` у больших значений `long`
- сравнивать `double` на точное равенство без допуска
- выбирать слишком маленький тип ради мнимой экономии

## Любопытный факт

Один шестнадцатеричный символ кодирует ровно 4 бита, поэтому hex-запись так удобна при работе с байтами.

## Практика

1. Создайте переменные всех примитивных типов.
2. Выведите их значения в консоль.
3. Попробуйте задать слишком большое значение для `byte` и посмотрите на ошибку компиляции.
