# 001. Установка JDK на Windows

## Что нужно установить

Для разработки на Java нужен `JDK`, а не только среда запуска. После установки должны работать команды `java` и `javac`.

## Шаги

1. Скачайте актуальный JDK для Windows с сайта Oracle или Adoptium.
2. Запустите установщик и завершите установку с параметрами по умолчанию.
3. Проверьте путь установки, обычно это папка вида `C:\Program Files\Java\jdk-...`.
4. Добавьте путь к папке `bin` в системную переменную `Path`, если установщик не сделал это автоматически.

## Проверка

Откройте новую консоль и выполните:

```powershell
java -version
javac -version
```

Если обе команды отрабатывают без ошибки, JDK установлен корректно.

## Любопытный факт

Многие новички путают `JRE` и `JDK`: первая нужна в основном для запуска, а вторая включает инструменты разработчика, поэтому для обучения почти всегда нужен именно `JDK`.
