# 001. Установка и настройка IntelliJ IDEA

## Что выбрать

Для учебных и обычных Java-проектов достаточно `IntelliJ IDEA Community Edition`.

## Быстрая настройка

1. Скачайте IntelliJ IDEA с сайта JetBrains.
2. Установите IDE с настройками по умолчанию.
3. При первом запуске создайте новый проект Java.
4. Укажите установленный `JDK` как `Project SDK`.
5. Создайте класс с методом `main` и запустите его.

## Минимум, который нужен в IDE

- выбран корректный `Project SDK`
- проект собирается без ошибок
- программа запускается кнопкой `Run`
- консоль показывает результат выполнения

## Любопытный факт

IntelliJ IDEA стала популярной во многом из-за сильного автодополнения и рефакторинга: IDE умеет не просто подсвечивать ошибки, а помогать безопасно менять структуру кода.
