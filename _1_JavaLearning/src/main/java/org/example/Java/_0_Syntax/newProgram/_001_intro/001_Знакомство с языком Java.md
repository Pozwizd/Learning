# 001. Знакомство с Java

## Коротко

Java - это язык программирования, который компилируется в байт-код и выполняется в JVM. За счет этого один и тот же код можно запускать на разных операционных системах.

## Где применяется Java

- backend и веб-сервисы
- корпоративные системы
- Android-разработка
- инструменты автоматизации
- настольные приложения и учебные проекты

## Что важно запомнить

- `JDK` - набор инструментов разработчика: компилятор, утилиты и все нужное для написания программ
- `JRE` - среда для запуска Java-приложений
- `JVM` - виртуальная машина, которая исполняет байт-код

## Любопытные факты

- Изначально язык назывался `Oak`.
- Публичный релиз Java состоялся 23 мая 1995 года.
- Название `Java` связано с кофе сорта Java, поэтому кофейная чашка стала частью узнаваемого образа языка.
- Идея `write once, run anywhere` сделала Java одним из самых заметных кроссплатформенных языков.

## Практика

1. Откройте `HelloWorld.java`.
2. Запустите программу из IDE.
3. Измените строку вывода на `Hello, YOUR_NAME!`.
4. При желании скомпилируйте файл через `javac` и запустите через `java`.
