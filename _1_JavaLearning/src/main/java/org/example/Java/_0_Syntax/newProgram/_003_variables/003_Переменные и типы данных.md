# 003. Переменные и типы данных

## Что такое переменная

Переменная - это именованная область памяти, в которой хранится значение определенного типа.

## Объявление и инициализация

```java
int count;
count = 10;

String name = "Java";
```

- объявление задает тип и имя
- инициализация задает первое значение

## Имена переменных

- используйте понятные имена: `userAge`, `totalPrice`
- стиль Java для переменных: `camelCase`
- имя не должно начинаться с цифры

## Приведение типов

### Неявное

```java
int number = 10;
long bigNumber = number;
```

### Явное

```java
double value = 9.8;
int rounded = (int) value;
```

## Техническая часть

- локальные переменные должны быть инициализированы до использования
- Java строго типизирована: тип нельзя игнорировать
- при приведении из `double` в `int` дробная часть отбрасывается

## Wrapper-классы

- `int -> Integer`
- `double -> Double`
- `char -> Character`

Они нужны, когда значение должно использоваться как объект.

## Типичные ошибки

- использовать переменную до присваивания значения
- путать `=` и `==`
- делать явное приведение и забывать о потере точности

## Практика

1. Создайте несколько переменных разных типов.
2. Выполните неявное и явное приведение.
3. Выведите результат и объясните, где данные изменились.
