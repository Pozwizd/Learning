# 04. Циклы

## Зачем нужны циклы

Циклы позволяют повторять один и тот же код много раз.

Без цикла пришлось бы писать одинаковые инструкции вручную.

## while

Цикл `while` выполняется, пока условие истинно.

```java
int i = 1;

while (i <= 5) {
    System.out.println(i);
    i++;
}
```

## do-while

Сначала выполняет код, потом проверяет условие.

```java
int i = 1;

do {
    System.out.println(i);
    i++;
} while (i <= 5);
```

Этот цикл выполнится хотя бы один раз.

## for

Самый частый цикл, когда известно количество повторений.

```java
for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}
```

Внутри `for` обычно есть:

- стартовое значение
- условие продолжения
- изменение счетчика

## for-each

Используется для перебора массива или коллекции.

```java
int[] numbers = {10, 20, 30};

for (int number : numbers) {
    System.out.println(number);
}
```

## break

Прерывает цикл полностью.

```java
for (int i = 1; i <= 10; i++) {
    if (i == 4) {
        break;
    }
    System.out.println(i);
}
```

## continue

Пропускает текущую итерацию.

```java
for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue;
    }
    System.out.println(i);
}
```

## Частые ошибки

- забыли увеличить счетчик в `while`
- получили бесконечный цикл
- ошиблись в условии на единицу: `<` вместо `<=`
