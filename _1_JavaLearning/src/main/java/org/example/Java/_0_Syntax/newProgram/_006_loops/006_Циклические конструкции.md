# 006. Циклические конструкции

## Зачем нужны циклы

Циклы позволяют многократно выполнять один и тот же блок кода, пока выполняется условие или пока не обработан весь набор данных.

## `while`

Проверяет условие перед каждой итерацией.

```java
int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}
```

## `do-while`

Тело выполнится минимум один раз.

```java
int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 5);
```

## `for`

Удобен, когда заранее известен счетчик.

```java
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
```

## `break` и `continue`

- `break` завершает цикл
- `continue` пропускает текущую итерацию

## Техническая часть

- следите за изменением счетчика, иначе получите бесконечный цикл
- условие завершения должно быть очевидным
- `for` обычно лучше подходит для перебора по индексу

## Типичные ошибки

- ошибка на единицу: `<=` вместо `<`
- забытый инкремент счетчика
- изменение переменной цикла в неожиданных местах

## Практика

1. Выведите числа от 1 до 10 через `for`.
2. Посчитайте сумму чисел от 1 до `n` через `while`.
3. Выведите только четные числа, используя `continue`.
