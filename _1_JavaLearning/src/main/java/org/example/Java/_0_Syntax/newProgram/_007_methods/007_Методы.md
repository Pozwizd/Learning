# 007. Методы

## Что такое метод

Метод - это именованный блок кода, который выполняет отдельную задачу. Методы уменьшают дублирование и делают программу проще для чтения и тестирования.

## Базовый синтаксис

```java
public static int sum(int a, int b) {
    return a + b;
}
```

Здесь:

- `public` - модификатор доступа
- `static` - метод можно вызвать без создания объекта
- `int` - тип возвращаемого значения
- `sum` - имя метода
- `(int a, int b)` - параметры

## Метод без возвращаемого значения

```java
public static void printHello() {
    System.out.println("Hello");
}
```

## Параметры и аргументы

- параметры объявляются в сигнатуре метода
- аргументы передаются при вызове метода

```java
int result = sum(2, 3);
```

## Область видимости

Переменные, объявленные внутри метода, существуют только внутри этого метода.

## Техническая часть

- тип возвращаемого значения должен совпадать с `return`
- если метод не `void`, все ветви должны что-то возвращать
- метод должен делать одну понятную задачу

## Типичные ошибки

- забыть `return`
- передать аргументы не того типа
- пытаться использовать локальную переменную вне метода

## Практика

1. Создайте метод `square`, который возвращает квадрат числа.
2. Создайте метод `printLine`, который печатает строку несколько раз.
3. Разбейте длинный `main` на 2-3 маленьких метода.
