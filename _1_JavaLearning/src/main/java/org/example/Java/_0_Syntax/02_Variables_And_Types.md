# 02. Переменные и типы данных

## Что такое переменная

Переменная - это именованное место в памяти, где хранится значение.

```java
int age = 25;
String name = "Anna";
double price = 19.99;
boolean isStudent = true;
```

## Формула объявления

```java
тип имя = значение;
```

Примеры:

```java
int number = 10;
char letter = 'A';
boolean active = false;
```

## Основные типы в Java

### Целые числа

- `byte`
- `short`
- `int`
- `long`

Обычно по умолчанию используют `int`.

### Дробные числа

- `float`
- `double`

Обычно используют `double`.

### Символ

- `char`

Пример:

```java
char grade = 'A';
```

### Логический тип

- `boolean`

Пример:

```java
boolean isOpen = true;
```

### Строка

- `String`

Пример:

```java
String city = "Kyiv";
```

## Разница между примитивами и String

Примитивы:

- хранят простое значение
- работают быстрее
- не имеют `null` по умолчанию в локальных переменных

`String`:

- это не примитив, а объект
- используется для текста
- поддерживает методы, например `length()`

## Изменение значения переменной

```java
int count = 1;
count = 2;
count = count + 3;
```

После этого `count` станет равен `5`.

## Константы

Если значение не должно меняться, используют `final`:

```java
final double PI = 3.14159;
```

## Приведение типов

### Неявное

Из меньшего типа в больший:

```java
int a = 10;
double b = a;
```

### Явное

Из большего типа в меньший:

```java
double price = 9.99;
int rounded = (int) price;
```

Здесь дробная часть отбросится.

## Полезные замечания

- `int x;` без присваивания нельзя использовать в локальном методе сразу
- `String` пишется с большой буквы, `int` и `double` - с маленькой
- для `long` часто добавляют `L`: `10000000000L`
- для `float` добавляют `f`: `3.14f`
