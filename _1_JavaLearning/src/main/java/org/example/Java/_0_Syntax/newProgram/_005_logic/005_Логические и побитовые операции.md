# 005. Логические и побитовые операции

## Логические операции

Они работают с `boolean` и используются в условиях.

- `&&` - логическое И
- `||` - логическое ИЛИ
- `!` - логическое НЕ

```java
boolean access = age >= 18 && hasTicket;
```

## Короткое замыкание

`&&` и `||` вычисляют правую часть только если это действительно нужно.

Это важно, когда правая часть дорогая по вычислениям или может вызвать ошибку.

## Побитовые операции

- `&` - побитовое И
- `|` - побитовое ИЛИ
- `^` - XOR
- `~` - инверсия
- `<<` - сдвиг влево
- `>>` - сдвиг вправо

```java
int mask = 0b1010;
int value = 0b1100;
int result = mask & value;
```

## Где это используется

- проверка флагов
- работа с правами доступа
- низкоуровневая обработка данных
- некоторые алгоритмы оптимизации

## Техническая часть

- `&` и `|` могут работать и с `boolean`, но без short-circuit
- сдвиг влево часто похож на умножение на степень двойки
- побитовые операции чаще применяют к `int` и `long`

## Типичные ошибки

- путать `&&` и `&`
- использовать побитовые операции без понимания двоичной формы
- писать магические маски без поясняющих имен

## Любопытный факт

Операция `XOR` активно используется не только в учебных примерах, но и в криптографии и контроле целостности данных.

## Практика

1. Напишите условие с `&&`, `||` и `!`.
2. Проверьте четность числа через побитовую операцию.
3. Сдвиньте число влево и вправо и сравните результат с обычным умножением и делением.
