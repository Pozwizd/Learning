# 009. Массивы

## Что такое массив

Массив хранит несколько значений одного типа под одним именем. Доступ к элементам идет по индексу.

## Создание массива

```java
int[] numbers = new int[5];
```

## Инициализация

```java
int[] numbers = {10, 20, 30, 40};
```

## Индексы

- индексация начинается с `0`
- последний элемент имеет индекс `length - 1`

```java
System.out.println(numbers[0]);
System.out.println(numbers[numbers.length - 1]);
```

## Перебор массива

```java
for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}
```

## Многомерные массивы

```java
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6}
};
```

В Java двумерный массив фактически является массивом массивов, поэтому строки могут быть разной длины.

## Техническая часть

- у массивов фиксированная длина
- выход за границы индекса вызывает `ArrayIndexOutOfBoundsException`
- для служебных операций часто используют класс `Arrays`

## Типичные ошибки

- начинать индекс с `1`
- писать `i <= array.length`
- забывать, что длину массива менять нельзя

## Любопытный факт

В Java "двумерный массив" не обязан быть прямоугольным: это позволяет создавать зубчатые структуры, где строки имеют разную длину.

## Практика

1. Создайте массив из 10 чисел и выведите его в обратном порядке.
2. Найдите минимум, максимум и среднее значение.
3. Создайте двумерный массив и выведите его в виде таблицы.
