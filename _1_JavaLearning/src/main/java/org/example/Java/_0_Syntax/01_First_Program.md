# 01. Первая программа на Java

## Что такое программа

Программа - это набор инструкций, которые компьютер выполняет по порядку.
В Java точка входа в программу обычно находится в методе:

```java
public static void main(String[] args)
```

Когда вы запускаете файл, JVM ищет именно этот метод и начинает выполнение с него.

## Минимальный пример

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}
```

## Что означает каждая часть

- `public class Main` - объявление класса с именем `Main`
- `main` - главный метод, с которого начинается выполнение
- `String[] args` - аргументы командной строки
- `System.out.println(...)` - вывод текста в консоль
- `;` - конец инструкции

## Базовые правила синтаксиса

- Java чувствительна к регистру: `main` и `Main` - разные имена
- каждая инструкция обычно заканчивается `;`
- код внутри метода пишется в фигурных скобках `{ }`
- имена переменных обычно пишут в `camelCase`
- имена классов обычно пишут с большой буквы: `User`, `Car`, `Main`

## Как читать код

Читай программу сверху вниз:

1. объявлен класс
2. внутри класса объявлен метод `main`
3. внутри `main` выполняется вывод строки

## Частые ошибки новичков

- забыли `;`
- написали `Main` вместо `main`
- забыли фигурную скобку
- имя файла не совпадает с именем публичного класса

## Практика

Попробуй самостоятельно:

1. вывести свое имя
2. вывести две строки подряд
3. заменить текст на число: `System.out.println(42);`
