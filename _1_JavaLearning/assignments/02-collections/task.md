# Assignment 02: Частотный словарь

## Цель
Научиться использовать `Map` для подсчёта частоты элементов.

## Условие
Реализовать метод, который считает количество повторений слов в списке.

## Шаги
1. Пройти по всем словам
2. Обновлять значение в `Map<String, Integer>`
3. Вернуть карту

## Критерии готовности
- Пустой вход даёт пустую map
- Повторяющиеся слова подсчитываются корректно

## Подсказки
- `getOrDefault`

## Доп. задание
- Верни топ-1 самое частое слово
