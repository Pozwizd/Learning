# Assignment 03: Фильтрация и нормализация

## Цель
Отработать Stream pipeline.

## Условие
Реализовать метод, который:
1. Удаляет пробелы по краям
2. Оставляет строки длиной >= 3
3. Переводит в upper-case

## Критерии готовности
- Корректная работа на пустом и непустом списке

## Подсказки
- `map(String::trim)`, `filter`, `map(String::toUpperCase)`

## Доп. задание
- Отсортируй результат по длине
