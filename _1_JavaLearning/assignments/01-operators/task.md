# Assignment 01: Калькулятор приоритета

## Цель
Закрепить приоритет операторов и условные выражения.

## Условие
Реализовать метод `evaluateScore(int base, int bonus, boolean hasPenalty)`.

## Шаги
1. Сложить `base` и `bonus`
2. Если `hasPenalty == true`, вычесть 3
3. Вернуть итог

## Критерии готовности
- Метод работает для всех трёх случаев: без штрафа, со штрафом, с нулевым бонусом

## Подсказки
- Используй тернарный оператор

## Доп. задание
- Добавь защиту от отрицательного результата (минимум 0)
